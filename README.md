# FIHSplits
Auto splitting tools and split trigger tools for Flipping is Hard!
<img width="500" height="200" alt="image" src="https://github.com/user-attachments/assets/9285e2f3-8395-4ab6-8440-44c59876abce" />

When you run the game with the DLL attached, it will automatically create a default Ruleset and Split times. The ruleset defines the triggers (Every trigger under a name will autosplit the name above once you enter one of them), and your Splits.txt will contain your best times for each split name (Split names must be exact and changing the ruleset does not automatically make more splits). If you need to remove a split time, set it to -1. If you need to change your time, enter the time in seconds. If you want to reset it, delete Splits.txt and restart the game. This is useful for making sure your splits are not affected by cheats or practice mods. To create a trigger, press "," (comma) and it will record a position at the player's center. Press "." (Period) to record a second position. Both positions will be used as corners to make a box. Then, that box will be output to a file named Split Debug trigger.txt, where you can paste it into Ruleset to add it (Make sure it's under the correct name). (There is no visualization for the box, you just need to remember where you pressed comma and where you press period).
